3 Math function

В калькуляторе реализованы следующие математические функции:

  1. Вычисляет косинус - cos(x)
  2. Вычисляет синус - sin(x)
  3. Вычисляет тангенс - tan(x)
  4. Вычисляет арккосинус - acos(x)
  5. Вычисляет арксинус - asin(x)
  6. Вычисляет арктангенс - atan(x)
  7. Считает квадратный корень - sqrt(x)
  8. Считает натуральный логарифм - ln(x)
  9. Считает обычный логарифм - log(x)
Введите выражение, нажмите "=" и получите ответ.
